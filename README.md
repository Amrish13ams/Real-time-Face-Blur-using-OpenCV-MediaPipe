# Real-time Face Blur using OpenCV & MediaPipe

This project detects faces in real-time using OpenCV and MediaPipe, applying a blur effect to anonymize detected faces.

## Requirements
- Python 3.x
- OpenCV
- MediaPipe

## Installation
```
pip install opencv-python mediapipe
```

## Usage
Run the script:
```
python face_blur.py
```
Press 'q' to exit.
